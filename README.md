## 系统介绍

> 前言

本项目是采用java+vue的前后端分离架构，主要是实现了一个本地的云盘服务，用户可以进行文件的上传下载。

> 技术栈

#### 前端：

Vue Node.js webpack ElementUI

#### 后端：

java SpringBoot Mybatis FastDFS Shiro SpringAOP

Mysql Redis 

RabbitMQ

> 前端项目打包运行

```shell
#启动
npm run dev
#打包
npm run build
```

后端项目打包

```shell
mvn clean package
```
